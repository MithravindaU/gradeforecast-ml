## 🚀 Overview

GradeForecast ML is a student performance prediction system that uses machine learning models to estimate academic scores based on study habits, attendance, and sleep patterns.

The project explores multiple regression techniques and compares their predictive performance to identify the most effective model for forecasting student marks.

## 📈 Models Implemented

### Linear Regression

Predicts marks using a linear relationship between features and academic performance.

### Decision Tree Regressor

Uses tree-based decision rules to estimate marks.

### Random Forest Regressor

Combines multiple decision trees to improve prediction accuracy and reduce overfitting.
